# 系统联调测试报告

## 一、测试概述

### 1.1 测试目的
验证家电超市管理系统在后端迁移后的功能完整性和数据一致性，确保所有业务流程在新的数据存储架构下正常运行。

### 1.2 测试范围
| 模块 | 功能点 |
|------|--------|
| 用户认证 | 登录、权限验证、会话管理 |
| 产品管理 | 增删改查、禁用/启用、阈值设置 |
| 库存管理 | 库存查询、调整、预警、流水记录 |
| 销售管理 | 订单创建、支付、退货、交付 |
| 采购管理 | 订单创建、收货、超期提醒 |
| 客户管理 | 客户信息管理、购买历史 |
| 财务管理 | 财务流水、利润计算、报表生成 |
| 报表统计 | 销售趋势、类别占比、供应商排名 |

### 1.3 测试环境
| 项目 | 描述 |
|------|------|
| 前端环境 | Vue 3 + Vite |
| 后端环境 | Flask + SQLite |
| API 地址 | http://localhost:5000 |
| 测试账号 | admin/123, cashier/123, purchaser/123 |
