## 使用说明

### 1. 数据迁移步骤

```bash
# 1. 在浏览器中打开 export_data.html，点击"导出数据"按钮
# 2. 将下载的 localStorage_data.json 放到项目根目录
# 3. 确保后端服务运行
cd backend
python app.py

# 4. 安装依赖并运行迁移脚本
npm install axios
node migrate.js
```

### 2. 自动化测试步骤

```bash
# 1. 确保后端服务运行
cd backend
python app.py

# 2. 运行测试脚本
node test_api.js
```

### 3. 手动测试步骤

```bash
# 1. 启动后端服务
cd backend
python app.py

# 2. 启动前端开发服务器
npm install
npm run dev

# 3. 访问 http://localhost:5173
# 4. 使用测试账号登录：admin/123, cashier/123, purchaser/123
```
